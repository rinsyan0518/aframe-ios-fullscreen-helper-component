### Installation

#### npm

Install via npm:

```bash
npm install git+ssh://git@github.com:rinsyan0518/aframe-ios-fullscreen-helper-component.git
```

Then require and use.

```js
require('aframe');
require('aframe-ios-fullscreen-helper-component');
```

```html
<body>
  <a-scene ios-fullscreen-helper>
  </a-scene>
</body>
```
